Readme of Lab4:

1. Baseline requirement:
	Agent motion:
		Predators chase prey when detected: you can observe it when viewing in the scene mode, the vision of predator is drawed by debugger
		prey try to avoid predators: the prey will run backward if it detect predators

	Agent vision:
    		Field of vision for predators, your choice of vision model for prey: both of them have field of vision
		
2. Additional features:
	Add fixed obstacles that the creatures cannot move through: all of them are trapped in a square area
	Add fixed obstacles that the creatures cannot see through: when prey runs behind the wall, predator will give up chasing
	Parameterize the agent's behavior or vision: you can change the fov and view distance in the inspector window
