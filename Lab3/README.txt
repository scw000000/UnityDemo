Readme of Lab3:

1. Baseline requirement:
	Physically based object movement:
		The objects will moved according constant gravity force

	Object management:
		The color of the object will change according to its age
		It will begin with red color, then gradually change to green, and blue as it approaches its maximum age

	Object emitter variation:
		The object properties will be set randomly according to predefined range
		These properties including age, radius, initial velocity
		Also, initial location will be set randomly inside a predefined sphere volume

		You can set these attributes in the script component of Game Object emitter
	
	Collision detection:
		Objects can collide with non-axis aligned plane

        Collision response:
		Objects respond to collisions by law of reflection

2. Additional features:
	The particles are displayed as billboard to increase performance
